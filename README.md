### Ocultar a barra de status no Android 
Para oclutar a barra de titulo do app devemos colocar o codigo a seguir no arquiv AndoridManifest.xml
```xml
    <application
        ...
        android:theme="@android:style/Theme.Holo.NoActionBar.Fullscreen" >
        ...
    </application>
```
